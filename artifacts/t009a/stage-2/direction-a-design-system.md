# T009A 第二阶段：方向 A / 方向 1

日期：2026-07-16

## 视觉原则

- 浅色、克制、专业、可信，现实素材图片优先。
- 品牌红只用于品牌、主操作、价格和关键焦点，不做促销式大面积铺色。
- 认证、授权和演示状态同时使用图标与文字，不只依赖颜色。
- 不使用渐变、霓虹、Emoji、手绘 SVG、CSS 图形或空白素材框。

## Token 摘要

| 类型 | 值 |
|---|---|
| Brand | `#C9342B` / hover `#A82720` / soft `#F8E9E7` |
| Surface | white / paper `#F7F6F3` |
| Text | ink `#1F1E1C` / muted `#6F6B66` |
| Border | `#E5E1DC` |
| Success / warning | `#4F7B45` / `#A8651B` |
| Radius | control 6px / panel and card 8px |
| Images | card, detail and thumbnail all 4:3 with `object-cover` |
| Icons | Phosphor Icons, line or duotone; success may use fill |

完整规则以 `docs/视觉设计基线.md` 第 8 节为本地事实来源。

## 六组代表页面

1. 首页：搜索、三类素材、认证/授权信任信息、精选素材。
2. 素材浏览/搜索：关键词、类型、认证和排序可交互；空结果可恢复。
3. 素材详情：4:3 主图与多角度切换、认证/授权层级、演示购买清单。
4. 登录/注册：邮箱/手机号切换、验证码流程、条款接受和安全说明。
5. 个人中心/上传：购买/下载/授权/上传/收益入口与演示上传表单；继续受真实角色守卫保护。
6. 管理后台：经营概览、待处理列表、模块导航和权限提示；继续受真实后台角色守卫保护。

## 明确不做

- 不接真实素材数据库、搜索 API、支付、COS、文件上传、审核、下载或认证服务。
- 不绕过登录、CSRF、邀请码和角色权限。
- 演示素材、订单、经营数字和上传反馈均明确标记为非真实。
