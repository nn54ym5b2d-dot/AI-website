# T009A 三套视觉方向生成提示

三套提示均使用 `comparison-content.md` 的固定内容，分别附带对应参考截图。生成方式：内置图像生成；每个方向独立生成一张 1440 × 1024 桌面首页 UI。

## A：专业创作市场

Use case: ui-mockup
Asset type: 1440 × 1024 desktop marketplace homepage UI
Primary request: 为中国大陆数字素材交易平台“源素库”设计一张真实、可落地、生产级桌面首页视觉稿。核心用户是 AI 视频、游戏、短剧广告和虚拟内容创作者。页面必须搜索优先、现实素材图片优先、认证与购买信息清楚。
Input images: Adobe Stock、站酷海洛、Fab 首页截图仅作为搜索层级、专业资产市场结构和商业可信感参考；不要复制品牌、Logo、促销或业务功能。
Style/medium: realistic product UI, not concept art; light professional creative marketplace; clear Chinese typography; restrained editorial commerce
Color palette: 白色、浅灰或轻暖灰表面，单一稳重品牌强调色；高可读对比
Composition/framing: 顶部简洁导航；大而清楚的首屏标题与搜索；三类入口；下方三张横向精选素材卡；充足留白；不使用浏览器或设备外框
