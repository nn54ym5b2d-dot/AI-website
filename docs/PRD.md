# PRD：源素库 / 虚拟素材网站 MVP

版本：v1.4
日期：2026-07-19
资料来源：已拆分进 `docs/` 的原始需求内容、`docs/决策记录.md` 和 Robert 后续明确指令；旧 Word 已由 Robert 主动删除

## 1. 产品定位

源素库是一个在中国大陆发布、主要面向中国大陆 AI 视频、游戏、短剧广告和虚拟内容制作方的数字素材交易平台。

平台提供以多角度照片为主的人物、物件/道具和场景素材，并通过认证上传、后台审核、付费下载、统一商业授权、授权记录、收益记录和合作方只读后台，建立可追踪、可管理、可商业化使用的数字素材流通体系。

## 2. 第一版形态

第一版范围：

- Web 网站
- 管理后台
- 可在电脑和手机浏览器打开

第一版不做：

- App
- 微信小程序
- 复杂社区功能
- 会员订阅
- 自动提现和复杂分账
- 企业授权系统
- 真实版权认证接口自动对接
- 实名认证（延期到 MVP 后）

## 3. 目标用户

### 上传者 / 创作者

任何个人或组织，收到邀请码并完成注册后，可以上传人物、物件、场景素材，并通过素材下载获得收益。

### 下载者 / 使用者

任何个人或组织，注册并登录后，可以购买和下载平台素材。

典型使用者包括：

- 个人 AI 视频创作者
- 游戏或虚拟内容制作方
- 短剧和广告制作团队
- 设计团队和创意工作室

### 证书版权 / 公司合作方

通过只读后台观察平台素材、认证、订单、授权、上传量、下载量和收益汇总，用于合作方了解平台经营情况；第一版合作方分成比例为 0%，后续如调整可使用这些汇总数据计算，不参与编辑、审核、退款、导出或系统设置。

## 4. 素材范围

第一版素材以图片/照片为主。

| 素材类型 | 内容说明 | 第一版规则 |
|---|---|---|
| 人物素材 | 真人形象多角度照片，如正面、侧面、背面、脸部、身体局部等 | 需提交必要证明材料；必须获得认证证书后才能上架 |
| 物件素材 | 家具、服装、装饰品、产品、生活用品等外观和细节照片 | 必须获得认证证书后才能上架 |
| 场景素材 | 室内空间、街景、商业空间、生活场景的整体和局部照片 | 必须获得认证证书后才能上架 |

后续扩展：

- 视频素材
- 3D 素材
- 素材包

## 5. 核心模块

| 模块 | 第一版内容 |
|---|---|
| 素材浏览系统 | 首页、分类页、搜索、筛选、素材详情页 |
| 用户系统 | 注册登录、用户身份、上传者资料、订单记录、下载记录、授权记录、收益记录 |
| 素材上传系统 | 图片/多角度照片上传、缩略图、预览图、文件信息、哈希值、审核状态 |
| 版权/认证系统 | 跳转政府官方网站认证，平台保存证书文件，并记录认证状态、证书编号和凭证 |
| 交易系统 | 一个订单可购买多个素材，记录支付状态 |
| 下载授权系统 | 付款后生成授权记录和默认 365 天下载资格，多角度原文件打包为一个 ZIP 交付 |
| 后台管理系统 | 素材审核、邀请码、认证证书录入/核验、订单、收益、用户、合作方只读账号和平台经营汇总 |

## 6. 核心流程

### 6.1 认证上传流程

1. 上传者登录后选择认证上传。
2. 填写素材名称和类型，说明可选填；上传原文件和预览图，人物素材还需上传必要证明材料。
3. 提交时按每份素材 10 元支付认证上传费。
4. 平台进行初审。
5. 初审未通过时，触发后台退款并原路退回认证上传费。
6. 初审通过后，引导上传者跳转政府官方网站进行认证。
7. 认证完成后，平台保存认证证书文件，并记录证书编号、凭证和认证状态。
8. 审核通过且获得认证证书后，素材可上架并显示认证状态。

### 6.2 购买下载流程

1. 购买用户浏览、搜索或筛选素材。
2. 进入素材详情页，查看价格、认证状态和预览信息。
3. 用户登录后创建订单并支付，一个订单可以包含多个素材。
4. 支付成功后，系统为订单内每个素材生成永久有效的授权记录；特殊撤销需记录原因。
5. 系统生成默认有效 365 天的下载入口；全部有效原文件打包成一个 ZIP，每次访问再换取短时 COS 签名 URL。
6. 收益按上传者 80%、平台 20%、外部观察员 0% 记录。

## 7. MVP 做与不做

| 第一版做 | 第一版不做 |
|---|---|
| Web 网站 + 管理后台 | App / 微信小程序 |
| 上传者邀请码注册/激活 | 无邀请码开放上传者注册 |
| 人物、物件/道具、场景图片素材 | 视频、3D、素材包 |
| 认证上传 | 真实版权认证接口自动对接 |
| 一个订单可购买多个素材 | 会员订阅、包月无限下载 |
| 授权记录 | 复杂企业授权系统 |
| 上传者收益记录 | 自动提现和复杂分账结算 |
| 外部观察员只读后台 | 合作方编辑或审核权限 |
| 仅人物素材提交必要证明材料 | 实名认证流程和实名资料存储 |

## 8. 成功标准

第一版可以认为达到 MVP 的标准，当以下流程能够完整跑通：

- 上传者通过邀请码注册并激活。
- 上传者提交人物、物件或场景素材。
- 平台能完成初审、认证状态记录和上架。
- 购买用户能浏览、搜索、筛选并购买多个素材。
- 支付成功后能生成永久授权记录和默认有效 365 天的下载入口。
- 上传者收益记录能被系统记录。
- 后台不同角色只能访问自己权限内的功能。
- 外部观察员只能只读查看平台上传量、下载量、收益汇总和合作方分成相关数据；第一版分成比例和金额为 0，且不能导出数据。
