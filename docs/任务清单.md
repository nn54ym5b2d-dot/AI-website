## 1. 状态说明

| 状态 | 含义 |
|---|---|
| Backlog | 后续再做 |
| Ready | 资料足够，可以开始 |
| In Progress | 正在执行 |
| Review | 等 Robert 检查 |
| Done | 已完成 |
| Blocked | 前置任务或必要条件尚未满足 |

本地任务清单和 GitHub Project #1 统一使用以上六种状态，不再使用 Project 默认的 `Todo`。任务开始时从 `Ready` 转为 `In Progress`，提交给 Robert 验收时转为 `Review`，验收并合并后转为 `Done`。
