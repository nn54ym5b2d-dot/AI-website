# 业务规则

版本：v1.2
日期：2026-07-13

## 1. 上传者规则

- 上传者必须通过邀请码注册或激活。
- 同一个上传者账号可以上传人物、物件/道具和场景素材。
- 第一版不开放无邀请码上传者注册。
- 企业用户第一版可先合并为购买用户；企业账户、企业授权、批量采购和对公支付放到后续版本。

## 2. 素材上传规则

- 第一版只处理图片/多角度照片素材。
- 每份素材需要保存原文件、预览图、缩略图、文件信息和哈希值。
- 上传者提交素材时需要填写素材名称和类型；说明可选填。
- 只有人物素材需要提交必要证明材料。
- 所有素材，包括人物/物件/场景，均需获得认证证书后才能上架。

## 3. 认证上传费规则

- 认证上传费默认每份素材 10 元。
- 10 元为后台可配置字段，后续可调整。
- 上传提交时支付认证上传费。
- 初审未通过时，当场触发后台退款并原路退回。
- 初审通过后进入政府官方网站认证流程。
- 认证上传费长期独立记录在 `certification_fee_charges`，不并入素材购买订单。

## 4. 审核和认证规则

| 阶段 | 规则 |
|---|---|
| 初审 | 平台检查素材基础质量、类型、信息完整度和明显违规情况 |
| 政府认证 | 初审通过后跳转政府官方网站完成政府背书认证 |
| 认证记录 | 平台保存证书文件，并记录证书编号、凭证、认证状态和认证时间 |
| 上架 | 审核通过且认证完成后才允许上架 |

当前待确认：

- 政府官方网站的具体名称、认证路径和证书样式。
- 认证异常后的重新提交规则。

## 5. 购买规则

- 第一版采用单次购买模式。
- 一个订单可以购买多个素材。
- 用户需登录后才能创建订单和支付。
- 支付成功后，订单内每个素材分别开放下载权限。
- 支付成功后，订单内每个素材分别生成授权记录。

## 6. 定价规则

- 平台统一定价，不由上传者自由定价。
- 第一版人物素材统一 50 元、场景素材统一 50 元、物件/道具素材统一 10 元。
- 后台预留定价字段，后续可修改。
- 上传者不能自行定价。

## 7. 授权规则

用户付费购买后，默认可用于商业内容制作。

授权记录至少包括：

- 购买用户
- 素材编号
- 订单编号
- 统一商业授权说明
- 授权时间
- 素材认证状态
- 必要使用限制

禁止行为至少包括：

- 转售素材原文件
- 冒充素材权利人
- 违法用途
- 侵权用途
- 平台规则明确不允许的其他用途

- 统一商业授权协议的正式法律文本。
- 是否区分个人商用和企业商用。
- 第一版授权永久有效，不设置自然到期时间；退款、违规或争议处理等特殊情况可撤销，并记录原因。

## 8. 下载规则

- 支付成功后生成默认有效 365 天的用户下载入口，后续可调整。
- 原始文件地址不能直接公开。
- 下载记录需要保存用户、素材、订单、下载时间和下载状态。
- 每次通过有效下载入口取文件时，由服务端生成短时 COS 签名 URL，不保存或公开长期原文件地址。

## 9. 收益规则

- 第一版素材购买收益按上传者 80%、平台 20% 分配，后续可调整。
- 外部观察员第一版分成比例为 0%，不参与素材购买收益分配，后续可调整。
- 支付成功后，上传者收益记录增加。
- 第一版只记录收益，不做自动提现和复杂分账结算。
- 提现/结算状态放到后续版本扩展。
- 退款后需要冲正对应的上传者和平台收益记录。

## 10. 状态规则

| 状态类型 | 建议状态 |
|---|---|
| 素材审核状态 | 草稿、待审核、审核通过、审核驳回 |
| 素材上架状态 | 未上架、已上架、已下架 |
| 认证状态 | 待认证、认证中、已认证、认证异常 |
| 订单状态 | 待支付、已支付、已取消、支付失败 |
| 授权状态 | 有效、已撤销 |
| 收益状态 | 无收益、有收益 |

## 11. 外部观察员规则

- 外部观察员是网站合作方，核心用途是查看平台整体经营数据；第一版分成比例为 0%，后续调整分成时以这些汇总数据为依据。
- 外部观察员只能只读查看允许范围内的数据。
- 外部观察员不能编辑、审核、上架、下架、退款、录入认证、修改价格、修改分成规则或管理账号。
- 外部观察员不允许导出任何后台数据，包括汇总数据和明细数据。
- 外部观察员可查看平台级汇总字段：统计周期、总上传素材数、新增上传素材数、已上架素材数、下架素材数、审核中素材数、已认证素材数、认证异常素材数。
- 外部观察员可查看素材类型汇总字段：人物素材数量、物件/道具素材数量、场景素材数量，各类型上传量、上架量、认证量和下载量。
- 外部观察员可查看下载汇总字段：总下载次数、付费下载次数、按日期/月份的下载量、按素材类型的下载量、下载趋势。
- 外部观察员可查看收益汇总字段：平台订单总额、已支付订单金额、退款金额、净收益、平台分成金额、上传者分成金额、合作方分成基数、合作方分成比例、合作方预计分成金额、合作方已结算金额、合作方待结算金额；第一版合作方比例和金额显示为 0。
- 外部观察员可查看有限订单/授权汇总字段：订单数量、已支付订单数量、退款订单数量、授权记录数量、授权状态汇总。
- 外部观察员不得查看用户手机号、邮箱、证件信息、支付账户、完整支付流水号、素材原文件地址、下载原始链接、内部操作日志、平台密钥或其他敏感信息。
