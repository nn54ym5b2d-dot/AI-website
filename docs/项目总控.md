## 3. 产品简述

源素库是面向中国大陆 AI 视频、游戏、短剧广告和虚拟内容制作方的数字素材交易平台。

第一版做 Web 网站 + 管理后台，支持人物、物件/道具、场景图片素材的认证上传、审核上架、付费购买、下载授权、收益记录和后台权限管理。

## 4. 第一版范围

| 类型 | 内容 |
|---|---|
| 第一版做 | Web 网站、管理后台、人物/物件/场景图片素材、邀请码上传者、认证证书保存、单次购买、授权记录、ZIP 素材包下载、收益记录、外部观察员只读看板 |
| 第一版不做 | App、微信小程序、会员订阅、自动提现、复杂分账、复杂社区、企业授权系统、真实版权认证接口自动对接、实名认证 |
| 首版默认规则 | 人物/场景 50 元，物件/道具 10 元；上传者 80%、平台 20%、观察员 0%；永久授权；下载入口 365 天；只按完整订单明细退款；仅人物要求证明材料 |
