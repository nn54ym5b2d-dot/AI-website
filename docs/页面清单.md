# 页面清单

版本：v1.3
日期：2026-07-13

## 1. T006 路由结构结论

T006 先建立结构优先的页面骨架，不做最终品牌视觉、不接入真实登录、支付、上传、数据库、COS 或认证服务。

公开网站主导航和首页显眼位置只展示面向大众或常规用户的入口：访客浏览、购买用户个人中心、上传者入口。管理后台和外部观察员不属于大众入口，只保留为页面底部低显眼内部入口。

第一版入口按角色分为：

| 角色/访问者 | 主要入口 | 说明 |
|---|---|---|
| 访客 | `/`、`/materials`、`/search`、`/materials/demo-asset` | 可浏览素材、搜索筛选、查看详情；购买前需要登录 |
| 购买用户 | `/account`、`/account/purchases`、`/account/downloads`、`/account/licenses`、`/checkout` | 查看购买、下载、授权记录；从素材详情进入订单支付 |
| 上传者 | `/upload`、`/account/uploads`、`/account/upload-status`、`/account/revenue`、`/account/uploader-profile` | 邀请码激活后上传素材，查看审核状态和收益 |
| 管理员 | `/admin` 及 `/admin/*` | 超级管理员、运营管理员、财务管理员共用后台入口；公开站点不显著展示 |
| 外部观察员 | `/observer` | 独立只读入口，查看平台上传量、下载量、收益汇总和合作方分成字段，不允许导出；公开站点不显著展示 |

## 2. 前台网站

| 页面 | 路由 | 第一版用途 | 优先级 |
|---|---|---|---|
| 首页 | `/` | 展示平台定位、素材入口、推荐分类、搜索入口和角色入口 | P0 |
| 分类页 | `/materials` | 按人物、物件/道具、场景浏览素材 | P0 |
| 搜索结果页 | `/search` | 支持关键词搜索和筛选 | P0 |
| 素材详情页 | `/materials/[assetId]` | 展示素材预览、价格、认证状态、用途说明和购买入口 | P0 |
| 登录页 | `/login` | 手机号、邮箱、微信组合登录入口 | P0 |
| 注册页 | `/register` | 购买用户注册；上传者需邀请码激活 | P0 |
| 上传页 | `/upload` | 上传者提交素材和证明材料 | P0 |
| 订单支付页 | `/checkout` | 展示订单素材、金额、支付方式和支付状态 | P0 |

| 个人中心首页 | `/account` | 购买用户和上传者的个人中心总入口 | P0 |

| 我的下载 | `/account/downloads` | 查看可下载素材和下载记录 | P0 |

## 4. 管理后台

| 页面 | 路由 | 第一版用途 | 优先级 |
|---|---|---|---|
| 后台首页 | `/admin` | 查看核心数据概览和待处理事项 | P0 |
| 素材审核 | `/admin/review` | 审核上传素材，驳回或通过初审 | P0 |
| 邀请码管理 | `/admin/invitations` | 创建、查看、禁用上传者邀请码 | P0 |
| 素材管理 | `/admin/assets` | 查看、上架、下架、编辑素材基础信息 | P0 |
| 用户管理 | `/admin/users` | 查看购买用户、上传者和后台账号 | P0 |
| 订单管理 | `/admin/orders` | 查看订单、订单明细和订单状态 | P0 |
| 支付记录 | `/admin/payments` | 查看支付和退款相关记录 | P0 |
| 授权记录 | `/admin/licenses` | 查看用户购买后生成的授权记录 | P0 |
| 收益记录 | `/admin/revenue` | 查看上传者收益记录 | P0 |
| 版权认证记录 | `/admin/certifications` | 录入或核验证书编号、凭证和认证状态 | P0 |
| 外部观察员账号管理 | `/admin/observer-accounts` | 创建和管理只读观察账号 | P1 |
| 操作日志 | `/admin/audit-logs` | 查看后台关键操作记录 | P1 |
| 系统设置 | `/admin/settings` | 配置价格、认证上传费、分成比例等 | P1 |

## 5. 外部观察员后台

| 页面 | 路由 | 第一版用途 | 优先级 |
|---|---|---|---|
| 外部观察员只读看板 | `/observer` | 展示平台上传量、下载量、订单金额、净收益、合作方分成相关数据 | P0 |

外部观察员入口暂定为独立 `/observer`。这样合作方入口更清晰；后续权限实现时，仍可在服务端复用后台权限模型。
