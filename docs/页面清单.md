# 页面清单

版本：v1.1  
日期：2026-06-16

## 1. 前台网站

| 页面 | 第一版用途 | 优先级 |
|---|---|---|
| 首页 | 展示平台定位、素材入口、推荐分类和搜索入口 | P0 |
| 分类页 | 按人物、物件/道具、场景浏览素材 | P0 |
| 搜索结果页 | 支持关键词搜索和筛选 | P0 |
| 素材详情页 | 展示素材预览、价格、认证状态、用途说明和购买入口 | P0 |
| 登录页 | 用户登录 | P0 |
| 注册页 | 购买用户注册；上传者需邀请码激活 | P0 |
| 上传页 | 上传者提交素材和证明材料 | P0 |
| 订单支付页 | 展示订单素材、金额和支付状态 | P0 |
| 下载页 | 展示已购买素材和限时下载入口 | P0 |

## 2. 个人中心

| 页面 | 第一版用途 | 优先级 |
|---|---|---|
| 我的购买 | 查看订单和购买记录 | P0 |
| 我的下载 | 查看可下载素材和下载记录 | P0 |
| 我的授权记录 | 查看已购买素材的授权记录 | P0 |
| 我的上传 | 上传者查看自己提交的素材 | P0 |
| 素材审核状态 | 上传者查看素材审核、认证和上架状态 | P0 |
| 收益记录 | 上传者查看素材收益记录 | P0 |
| 上传者资料 | 上传者维护基础资料 | P1 |

## 3. 管理后台

| 页面 | 第一版用途 | 优先级 |
|---|---|---|
| 后台首页 | 查看核心数据概览和待处理事项 | P0 |
| 素材审核 | 审核上传素材，驳回或通过初审 | P0 |
| 邀请码管理 | 创建、查看、禁用上传者邀请码 | P0 |
| 素材管理 | 查看、上架、下架、编辑素材基础信息 | P0 |
| 用户管理 | 查看购买用户、上传者和后台账号 | P0 |
| 订单管理 | 查看订单、订单明细和订单状态 | P0 |
| 支付记录 | 查看支付和退款相关记录 | P0 |
| 授权记录 | 查看用户购买后生成的授权记录 | P0 |
| 收益记录 | 查看上传者收益记录 | P0 |
| 外部观察员数据看板 | 只读展示平台上传量、下载量、订单金额、净收益和合作方分成相关数据 | P0 |
| 版权认证记录 | 录入或核验证书编号、凭证和认证状态 | P0 |
| 外部观察员账号管理 | 创建和管理只读观察账号 | P1 |
| 操作日志 | 查看后台关键操作记录 | P1 |
| 系统设置 | 配置价格、认证上传费、分成比例等 | P1 |

## 5. 页面范围待确认

- 第一版首页是否需要完整品牌视觉，还是先做功能型首页。
- 素材详情页是否展示水印预览图。
- 上传页是否需要分人物、物件、场景三套表单。
- 后台首页需要展示哪些统计指标。
- 外部观察员后台是否独立入口，还是复用管理后台但限制权限。
- 外部观察员数据看板的统计周期默认按日、周、月还是自定义区间。
