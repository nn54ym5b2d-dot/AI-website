# 页面清单

版本：v3.6
日期：2026-07-22

## 1. T006 路由结构结论

T006 先建立结构优先的页面骨架，不做最终品牌视觉、不接入真实登录、支付、上传、数据库、COS 或认证服务。

公开网站主导航和首页显眼位置只展示面向大众或常规用户的入口：访客浏览、购买用户个人中心、上传者入口。管理后台和外部观察员不属于大众入口，只保留为页面底部低显眼内部入口。

第一版入口按角色分为：

| 角色/访问者 | 主要入口 | 说明 |
|---|---|---|
| 访客 | `/`、`/materials`、`/search`、`/materials/[assetId]` | 可浏览素材、搜索筛选、查看详情；购买前需要登录 |
| 购买用户 | `/account`、`/account/purchases`、`/account/downloads`、`/account/licenses`、`/checkout` | 查看购买、下载、授权记录；从素材详情进入订单支付 |
| 上传者 | `/upload`、`/account/uploads`、`/account/upload-status`、`/account/revenue`、`/account/uploader-profile` | 邀请码激活后上传素材，查看审核状态和收益 |
| 管理员 | `/admin` 及 `/admin/*` | 超级管理员、运营管理员、财务管理员共用后台入口；公开站点不显著展示 |
| 外部观察员 | `/observer` | 独立只读入口，查看平台上传量、下载量、收益汇总和合作方分成字段；首版分成比例为 0，不允许导出；公开站点不显著展示 |

## 9. T011 上传页实现状态

- `/upload` 已从视觉占位升级为真实 PostgreSQL 草稿工作区：人物素材使用独立表单，物件/道具与场景共用一套表单。
- 上传者可以创建草稿、选择最近草稿继续编辑、添加标签，并选择原文件；人物表单要求必要证明材料，物件/道具与场景表单提供“证明材料（选填）”入口，说明为“如有来源、版权或所有权证明，可选择上传。”
- 文件选择后由浏览器计算 SHA-256，再调用本地测试 provider 完成上传意图、幂等确认和衍生任务；页面明确提示当前只验证元数据、不保存真实文件、未接 COS。
- 页面展示原文件、人物证明、预览图和缩略图计数及处理状态；只有必需文件与水印衍生图全部就绪后才能提交。
- 点击最终“检查并提交”成功后，当前编辑界面切换为独立的“素材资料已提交 / 等待支付认证上传费”结果页；提供“查看我的上传”和“继续上传素材”入口。在真实支付回调前不显示为已进入审核；支付成功后的“提交成功，等待审核”页面留给 T013 完成。
