# 数据库设计草案

版本：v1.2  
日期：2026-06-16  
状态：草案，技术方向、腾讯云 COS 和腾讯云生产部署已确认

## 1. 设计说明

本文件先定义核心数据对象，方便后续开发时转成具体数据库表。字段名暂用英文，说明用中文。

## 2. 核心表

### users

平台用户基础表。

| 字段 | 说明 |
|---|---|
| id | 用户 ID |
| name | 用户名称 |
| email | 邮箱，用于邮箱登录 |
| phone | 手机号，用于手机号登录 |
| primary_login_method | 主要登录方式：phone、email、wechat |
| role | 基础角色：buyer、uploader、admin 等 |
| status | 正常、禁用等 |
| created_at | 创建时间 |
| updated_at | 更新时间 |

### auth_identities

第三方或多方式登录身份表。

| 字段 | 说明 |
|---|---|
| id | 登录身份 ID |
| user_id | 关联用户 |
| provider | phone、email、wechat |
| provider_user_id | 第三方或登录方式下的唯一 ID |
| verified_at | 验证时间 |
| created_at | 创建时间 |

### uploader_profiles

上传者资料表。

| 字段 | 说明 |
|---|---|
| id | 上传者资料 ID |
| user_id | 关联用户 |
| invite_code_id | 关联邀请码 |
| display_name | 上传者展示名称 |
| verification_status | 上传者资料状态 |
| created_at | 创建时间 |

### invite_codes

邀请码表。

| 字段 | 说明 |
|---|---|
| id | 邀请码 ID |
| code | 邀请码 |
| status | 未使用、已使用、禁用 |
| created_by | 创建管理员 |
| used_by | 使用者 |
| used_at | 使用时间 |

### assets

素材主表。

| 字段 | 说明 |
|---|---|
| id | 素材 ID |
| uploader_id | 上传者 |
| asset_type | person、object、scene |
| title | 素材标题 |
| description | 素材说明 |
| tags | 分类/搜索标签 |
| review_status | 草稿、待审核、审核通过、审核驳回 |
| listing_status | 未上架、已上架、已下架 |
| certification_status | 待认证、认证中、已认证、认证异常 |
| price | 当前售价 |
| created_at | 创建时间 |
| updated_at | 更新时间 |

### asset_files

素材文件表。

| 字段 | 说明 |
|---|---|
| id | 文件 ID |
| asset_id | 关联素材 |
| file_type | 原文件、预览图、缩略图、证明材料、认证证书文件 |
| storage_path | 腾讯云 COS 存储路径 |
| file_hash | 文件哈希 |
| file_size | 文件大小 |
| mime_type | 文件类型 |
| created_at | 创建时间 |

说明：证明材料只有人物素材必需；认证证书文件在认证完成后必须保存。

### certifications

认证记录表。

| 字段 | 说明 |
|---|---|
| id | 认证记录 ID |
| asset_id | 关联素材 |
| status | 待认证、认证中、已认证、认证异常 |
| certificate_no | 证书编号 |
| certificate_file | 认证证书文件路径，平台必须保存 |
| certificate_snapshot | 认证凭证截图或补充材料 |
| verified_by | 核验管理员 |
| verified_at | 核验时间 |
| notes | 备注 |

### orders

订单表。

| 字段 | 说明 |
|---|---|
| id | 订单 ID |
| user_id | 购买用户 |
| status | 待支付、已支付、已取消、支付失败 |
| total_amount | 订单总金额 |
| paid_at | 支付时间 |
| created_at | 创建时间 |

### order_items

订单明细表。

| 字段 | 说明 |
|---|---|
| id | 明细 ID |
| order_id | 关联订单 |
| asset_id | 关联素材 |
| price | 购买时价格 |
| uploader_id | 上传者 |

### authorization_records

授权记录表。

| 字段 | 说明 |
|---|---|
| id | 授权 ID |
| user_id | 购买用户 |
| order_id | 关联订单 |
| asset_id | 关联素材 |
| license_text | 统一商业授权说明 |
| status | 有效、已撤销、已过期 |
| granted_at | 授权时间 |

### downloads

下载记录表。

| 字段 | 说明 |
|---|---|
| id | 下载记录 ID |
| user_id | 下载用户 |
| asset_id | 下载素材 |
| order_id | 关联订单 |
| download_url_expires_at | 限时链接过期时间 |
| downloaded_at | 下载时间 |

### revenue_records

收益记录表。

| 字段 | 说明 |
|---|---|
| id | 收益记录 ID |
| uploader_id | 上传者 |
| asset_id | 素材 |
| order_item_id | 订单明细 |
| gross_amount | 素材售价 |
| uploader_amount | 上传者收益 |
| platform_amount | 平台收益 |
| status | 已记录、已冲正、待结算等 |
| created_at | 创建时间 |

### payments

支付记录表。

| 字段 | 说明 |
|---|---|
| id | 支付记录 ID |
| order_id | 关联订单 |
| provider | 支付服务商 |
| amount | 支付金额 |
| status | 待支付、成功、失败 |
| provider_trade_no | 第三方交易号 |
| created_at | 创建时间 |

说明：`provider` 第一版支持 `wechat_pay` 和 `alipay`。

### refunds

退款记录表。

| 字段 | 说明 |
|---|---|
| id | 退款记录 ID |
| payment_id | 关联支付 |
| order_id | 关联订单，可为空 |
| asset_id | 认证上传费退款时关联素材 |
| amount | 退款金额 |
| reason | 退款原因 |
| status | 待处理、成功、失败 |
| created_at | 创建时间 |

### admin_roles

后台角色表。

| 字段 | 说明 |
|---|---|
| id | 角色 ID |
| user_id | 用户 |
| admin_role | super_admin、operator、finance、observer |
| status | 正常、禁用 |

### audit_logs

操作日志表。

| 字段 | 说明 |
|---|---|
| id | 日志 ID |
| actor_id | 操作者 |
| action | 操作类型 |
| target_type | 被操作对象类型 |
| target_id | 被操作对象 ID |
| metadata | 操作补充信息 |
| created_at | 创建时间 |

### platform_metric_snapshots

平台经营汇总快照表，供后台和外部观察员只读查看。

| 字段 | 说明 |
|---|---|
| id | 快照 ID |
| period_type | day、week、month、custom |
| period_start | 统计开始时间 |
| period_end | 统计结束时间 |
| total_uploaded_assets | 总上传素材数 |
| new_uploaded_assets | 新增上传素材数 |
| listed_assets | 已上架素材数 |
| delisted_assets | 已下架素材数 |
| pending_review_assets | 审核中素材数 |
| certified_assets | 已认证素材数 |
| certification_exception_assets | 认证异常素材数 |
| person_asset_count | 人物素材数量 |
| object_asset_count | 物件/道具素材数量 |
| scene_asset_count | 场景素材数量 |
| total_downloads | 总下载次数 |
| paid_downloads | 付费下载次数 |
| paid_order_count | 已支付订单数 |
| gross_order_amount | 平台订单总额 |
| refund_amount | 退款金额 |
| net_revenue | 净收益 |
| platform_share_amount | 平台分成金额 |
| uploader_share_amount | 上传者分成金额 |
| created_at | 创建时间 |

### observer_share_records

外部观察员合作分成记录表。

| 字段 | 说明 |
|---|---|
| id | 分成记录 ID |
| observer_user_id | 外部观察员账号 |
| period_start | 统计开始时间 |
| period_end | 统计结束时间 |
| share_base_amount | 合作方分成基数 |
| share_rate | 合作方分成比例 |
| expected_share_amount | 合作方预计分成金额 |
| settled_share_amount | 合作方已结算金额 |
| pending_share_amount | 合作方待结算金额 |
| status | 待确认、待结算、已结算、已冲正 |
| created_at | 创建时间 |
