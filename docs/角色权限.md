# 角色权限

版本：v1.4
日期：2026-07-15

## 1. 角色列表

| 角色 | 说明 |
|---|---|
| 购买用户 | 平台素材购买和下载方 |
| 上传者 | 通过邀请码注册/激活的平台素材提供方 |
| 超级管理员 | Robert 或核心负责人，拥有全部后台权限 |
| 运营管理员 | 日常运营、内容审核和认证信息处理 |
| 财务管理员 | 财务、支付、退款和收益管理 |
| 外部观察员 | 网站合作方，只读查看平台上传量、下载量、收益汇总和合作方分成相关数据 |

## 3. 关键边界

### 超级管理员

- 拥有全部后台管理权限。
- 可以管理用户、素材、订单、支付、认证、收益、角色和系统设置。
- 超级管理员身份不会自动赋予购买或上传业务身份；需要购买/下载时还必须具备 `buyer`，需要上传时还必须具备 `uploader`。

### 运营管理员

- 可以审核素材、上架/下架、管理分类、查看订单和授权记录、录入或核验认证信息、处理用户问题。
- 不能修改角色和系统级设置。

### 财务管理员

- 可以查看订单、支付记录、收益记录和退款记录。
- 可以按后台流程发起和处理退款，并留下操作日志。
- 不能审核素材、修改素材内容、录入认证信息、修改价格或修改系统设置。

### 外部观察员

- 只能只读查看平台经营汇总和合作方分成相关数据。
- 可以查看整个平台网站上传量、下载量、收益情况；第一版观察员分成比例为 0%，后续调整时以这些汇总数据为依据。
- 可以查看统计周期、素材类型汇总、上传量、上架量、认证量、下载量、订单金额、退款金额、净收益、合作方分成基数、合作方分成比例、预计分成、已结算和待结算金额；第一版合作方比例和金额为 0。
- 不能编辑、审核、上架、下架、退款、录入认证、修改价格、修改分成规则、管理账号或导出数据。
- 不能查看用户手机号、邮箱、证件信息、支付账户、完整支付流水号、素材原文件地址、下载原始链接、内部操作日志或平台密钥。
