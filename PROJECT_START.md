## 2. 项目一句话说明

源素库是一个面向 AI 视频、游戏、短剧广告和虚拟内容制作方的数字素材交易平台。第一版做 Web 网站和管理后台，支持人物、物件/道具、场景图片素材的认证上传、审核上架、单次购买、下载授权、收益记录和后台权限管理。
